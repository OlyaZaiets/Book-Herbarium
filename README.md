# 📚 Book Herbarium 🌿

**Book Herbarium** is a full-stack web application for managing your personal reading collection and tracking your literary journey. It lets users organize books, store details, and build a structured digital library — like a botanical herbarium, but for stories.

---

## 🚀 Live Demo
# [**Live Demo – Book Herbarium **](https://book-herbarium.vercel.app/en) 🎬

---

## ✨ Features

- 📖 Add, edit, and manage books in your personal collection  
- 🔐 Authentication system with protected routes  
- 🗄️ PostgreSQL database with Prisma ORM  
- ⚡ Modern full-stack architecture using Next.js  
- 🧩 Component-based UI for scalability and reuse  
- 🖼️ Book cover storage and rendering  
- 🧭 Dynamic routing between pages  
- 🧠 Ready for AI-powered features (summaries)

---

## 🛠 Tech Stack

**Frontend:** Next.js, React, TypeScript  
**Backend:** Next.js API routes/server actions  
**Database:** PostgreSQL (Neon serverless)  
**ORM:** Prisma  
**Auth:** NextAuth  
**Icons:** Lucide  
**Linting & Formatting:** ESLint, Prettier
